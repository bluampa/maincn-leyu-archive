# maincn-leyu-archive

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面均为静态内容，不依赖于任何特定服务器或后端服务，可直接在浏览器中打开查看。

仓库名称中的 “maincn-leyu-archive” 仅作为标识，不代表与任何具体网站或服务有关联。

## 目录说明

- `pages/` — 存放所有归档的 HTML 页面文件。每个文件均为独立页面，可单独访问。
- `assets/` — 存放页面所需的公共资源文件（如 CSS、JavaScript、图片等）。若页面没有外部依赖，此目录可能为空。
- `docs/` — 可选目录，用于存放与页面使用或归档说明相关的附加文档。
- `index.html` — 仓库根目录下的索引页面，用于快速浏览或导航到各归档页面。

## 页面归档说明

- 所有归档页面均为原始 HTML 文件，未经过压缩或混淆处理。
- 页面内容可能包含文本、图片、表格、交互元素等，但不包含任何动态数据请求。
- 部分页面可能引用外部资源（如字体、图标库），这些资源通过 CDN 或本地文件加载。
- 页面文件名与内容主题一致，便于识别。

## 使用方式

1. 直接点击 `pages/` 目录下的 HTML 文件，在浏览器中查看。
2. 若仓库启用了 GitHub Pages，也可通过对应的 URL 访问根目录或 `pages/` 下的页面。
3. 可将整个仓库克隆到本地，在本地环境中打开页面文件。
